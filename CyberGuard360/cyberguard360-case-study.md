# CyberGuard360

## About CyberGuard360
CyberGuard360 is a web-based Cyber Security platform that provides breach prevention, employee cybersecurity education, compliance tracking, documentation management, and advanced threat-monitoring capabilities. The system supports a large user ecosystem with:

- **300+ Partners (MSPs)**
- **1400+ Client organizations**
- **15,000+ end users**

The platform is designed to help organizations identify risks, train employees, manage policies, monitor dark-web threats, and maintain regulatory compliance.

## Key Responsibilities & Features Implemented

### 1. Training & Compliance
- Developed modules for cybersecurity training programs.
- Implemented logic for tracking employee training, status, scores, and compliance reports.
- Automated reminder & notification systems.

### 2. Risk Assessments
- Built interactive assessment tools for organizations to evaluate cybersecurity posture.
- Integrated dynamic scoring with weighted risk parameters.
- Generated risk summary reports & dashboards.

### 3. Policy Management
- Created policy creation, distribution, and acknowledgment workflows.
- Implemented version management and audit logs for compliance tracking.

### 4. Phishing Simulation
- Integrated phishing simulation functionality using **GoPhish**.
- Built custom APIs to manage phishing campaigns, templates, user groups, and reporting.
- Processed campaign results and displayed analytics dashboards.

### 5. Dark Web Monitoring
- Integrated APIs for detecting compromised credentials on dark web databases.
- Implemented automated alerts, reporting, and tracking for exposed user accounts.

### 6. Documentation Module
- Designed and developed a centralized documentation management system.
- Enabled secure storage, version control, and role-based access.

### 7. Microsoft Azure Active Directory Integration
- Implemented authentication and user provisioning using **Azure AD (SSO)**.
- Configured OAuth2/OpenID Connect & handled role-mapping for enterprise users.
